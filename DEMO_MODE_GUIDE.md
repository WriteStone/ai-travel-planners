# 演示模式使用说明

## 🎉 现在可以直接体验应用了！

由于尚未配置 Supabase 数据库，应用已自动启用**演示模式**。

## ✨ 演示模式特点

### ✅ 可用功能
- ✅ **用户注册/登录** - 输入任意邮箱密码即可
- ✅ **行程规划** - 使用表单或语音创建行程
- ✅ **行程管理** - 查看、编辑、删除行程
- ✅ **费用追踪** - 记录和管理旅行费用
- ✅ **数据持久化** - 数据保存在浏览器本地存储

### ⚠️ 限制说明
- 数据仅保存在**当前浏览器**中
- 清除浏览器数据会丢失所有信息
- 无法跨设备同步
- AI 生成功能需要配置 OpenAI API 密钥
- 地图功能需要配置高德地图 API 密钥

## 🚀 快速开始

### 1. 注册/登录
访问 http://localhost:3000/auth/register
- 输入**任意邮箱和密码**（演示模式下无需真实邮箱）
- 点击注册即可进入系统

### 2. 创建行程
1. 点击「创建行程」按钮
2. 填写旅行信息：
   - 目的地：例如「东京」
   - 日期：选择出发和返回日期
   - 预算：例如「10000」
   - 人数：选择旅行人数
   - 偏好：勾选您的兴趣（美食、文化、自然等）
3. 点击「生成智能行程」

### 3. 查看行程
- 在仪表盘查看所有行程
- 点击行程卡片查看详情
- 切换「行程详情」、「地图导航」、「费用管理」标签

### 4. 管理费用
- 在行程详情页切换到「费用管理」标签
- 添加费用记录
- 查看预算使用情况

## 🔧 配置完整功能

要使用完整功能（云端同步、AI 生成、地图导航等），需要配置以下 API：

### 方式一：在应用中配置（推荐）
1. 访问 http://localhost:3000/settings
2. 输入各个 API 密钥
3. 点击保存

### 方式二：编辑环境变量
编辑 `f:\work\.env.local` 文件：

```env
# Supabase - 用户认证和数据存储
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key

# OpenAI - AI 行程生成
OPENAI_API_KEY=sk-your-api-key

# 高德地图 - 地图功能
NEXT_PUBLIC_AMAP_KEY=your-amap-key
```

### 获取 API 密钥

详细步骤请查看：`f:\work\API_KEYS_GUIDE.md`

**快速链接：**
- [Supabase](https://supabase.com) - 免费注册，创建项目
- [OpenAI](https://platform.openai.com) - 需要付费 API
- [高德地图](https://lbs.amap.com) - 免费申请

## 💡 提示

### 演示模式下的数据
演示模式数据存储在浏览器的 localStorage 中：
- `ai-travel-planner-demo-user` - 用户信息
- `ai-travel-planner-demo-trips` - 行程数据
- `ai-travel-planner-demo-expenses` - 费用数据

### 清除演示数据
打开浏览器控制台（F12），执行：
```javascript
localStorage.removeItem('ai-travel-planner-demo-user')
localStorage.removeItem('ai-travel-planner-demo-trips')
localStorage.removeItem('ai-travel-planner-demo-expenses')
localStorage.removeItem('ai-travel-planner-demo-mode')
```

### 切换到正式模式
配置真实的 Supabase 密钥后，应用会自动切换到正式模式，数据将保存到云端数据库。

## ❓ 常见问题

### Q: 为什么注册不了？
A: 请确保：
1. 应用正在运行（http://localhost:3000）
2. 页面上显示"演示模式"提示
3. 刷新页面重试

### Q: 数据会丢失吗？
A: 演示模式数据保存在浏览器中，清除浏览器数据或更换浏览器会丢失。要永久保存，请配置 Supabase。

### Q: AI 功能为什么不能用？
A: AI 行程生成需要 OpenAI API 密钥。您可以：
1. 在设置页面配置 API 密钥
2. 或手动填写行程信息（不使用 AI 生成）

### Q: 地图显示不出来？
A: 需要配置高德地图 API 密钥。访问设置页面配置即可。
